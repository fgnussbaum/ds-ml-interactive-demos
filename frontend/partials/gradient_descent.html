What you are minimizing: L(θ) = 1/n Σ (θ·xi + b − yi)2 — a parabola in θ (the curve on the left). Setting dL/dθ = 0 and solving gives the exact minimum in one step: θ* = Σ xiyi / Σ xi2 (green ★).
Gradient descent finds the same point iteratively. While this is slower here, it generalises to any differentiable loss with no closed form (neural networks, robust regression, …).
